    Rust VSCode插件 #

    1. rust-analyzer #

    实时编译和分析Rust代码，提示代码中的错误，并对类型进行标注。也可以使用官方的rust插件取代

    2. rust syntax #

    为代码提供语法高亮

    3. crates #

    帮助分析当前项目的依赖是否是最新的版本

    4. better toml #

    Rust使用toml做项目的配置管理，better toml可以语法高亮，并展示toml文件中的错误

    5. rust test lens #

    可以快速运行某个Rust测试

    6. Tabnine #
